# 🌙 安眠匣 (SleepHaven)

**安眠匣**是一款基于 Vue3 + uni-app 开发的智能睡眠监测与健康管理应用。它致力于通过轻量级的监测手段，帮助用户量化睡眠质量，管理“睡眠负债”，并提供基于 AI 的专业改善建议。无论是午间小憩还是夜间深睡，安眠匣都是你枕边的健康守护者。

---

## ✨ 核心功能

### 📊 智能睡眠监测
- **自动时段识别**：根据当前时间自动判断是“午休小憩”还是“夜间睡眠”模式。
- **后台持续计时**：支持锁屏运行，精准记录睡眠时长。
- **中途唤醒检测**：自动监测睡眠过程中的亮屏/切出行为，作为睡眠连续性的评估依据。
- **有效睡眠判定**：用户可自定义“最低有效时长”，避免误触导致的无效数据记录。

### 💤 睡眠负债管理
- **负债累计算法**：基于每日目标睡眠时长（默认8小时），自动计算累计睡眠赤字或盈余。
- **可视化状态**：通过颜色编码（红/橙/绿）直观展示当前的疲劳程度。
- **手动重置**：支持在补觉后手动清零负债，保持数据准确性。

### 🤖 AI 睡眠专家
- **深度数据分析**：集成硅基流动 (SiliconFlow) 大模型接口，结合睡眠时长、评分及唤醒次数生成个性化建议。
- **Markdown 报告**：以结构化的图文形式展示 AI 建议，涵盖入睡指导、环境优化及作息调整方案。
- **异步非阻塞**：AI 分析在后台异步进行，不影响用户正常操作体验。

### 🎨 沉浸式体验
- **多主题适配**：支持 **浅色**、**深色** 及 **跟随系统** 三种模式，完美适配夜间使用场景，保护视力。
- **响应式设计**：一套代码编译至 H5、微信小程序、App 等多端，界面流畅统一。
- **数据持久化**：所有历史记录、设置偏好均本地存储，隐私安全，无网可用。

## 📱 使用场景

1.  **午间快充**：下午1点，点击“开始小憩”，20分钟后醒来结束，AI 提示你“短时高效，下午精力充沛”。
2.  **夜间深睡**：晚上11点，开启“夜间睡眠”，次日早上7点结束。系统计算出昨晚睡眠不足，累计负债 +30 分钟，并建议今晚提前 30 分钟入睡。
3.  **长期追踪**：在“历史统计”中查看过去一个月的睡眠趋势，调整作息规律。

## 🚀 快速开始

### 1. 环境准备
确保已安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html) 编辑器。

### 2. 运行项目
1.  将源代码导入 HBuilderX。
2.  点击菜单栏 **运行** -> **运行到浏览器** (推荐 Chrome)。
3.  或在 `manifest.json` 中配置小程序 AppID 后，运行到微信开发者工具。

### 3. 配置 AI (可选)
默认代码中已内置测试用 API Key。如需长期使用或更换模型，请在 `pages/index/index.vue` 的 `<script setup>` 顶部修改配置：

```javascript
const API_KEY = '你的_SiliconFlow_API_Key';
const MODEL_NAME = 'deepseek-ai/DeepSeek-V3.2'; // 可替换为其他模型
```
